How the Predictor works

The goal of this project is to detect COVID-19 pneumonia patients automatically using digital chest X-ray images, while optimizing the detection accuracy by using a well-trained neural network model.

Dataset used for training purposes-

In most of the previous research and work done in this particular field, the main problem was that the data or dataset collected and used were either not adequete enough, or required further information and insights.

To mitigate this problem and to get better and more accurate results, we were careful to collect proper covid-19 positive sample images from this open-sourced data mentioned here.

This data was collected by Joseph Paul Cohen, who is the director of the Institute for Reproducible Research, a US non-profit which operates ShortScience.org and Academic Torrents. He is also a researcher and a pragmatic engineer.

The negative examples (bacterial and viral pneumonia not caused by covid infections) were collected from the publicly available and verified dataset from Kaggle.

Details about the model-

Before feeding to the neural network, the training data is augmented to create even more training sample by shearing the image (think of it like stretching the image horizontally), using only some parts of the image by zooming it to certain regions, and horizontal flipping. The test data do not undergo these changes.
Both the training and test data are rescaled by normalizing the pixel data, and fitted to a shape of dimensions 224x224x3.

A convolutional neural network (henceforth referred to as CNN) architecture is used for designing the deep-learning model. The model has 3 blocks of layers, and each block has a convolution layer followed by a pooling and a drop-out layer. This is then followed by a flatten and then a dense which later outputs a binary value- '1' or '0' as to whether it detects presence of covid in a given sample or not. The model uses binary cross-entropy for calculating loss, and uses the Adam optimizer.
